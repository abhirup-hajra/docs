A versatile, general-purpose model that balances speed and intelligence for everyday use, ranging from writing to problem-solving.

<Info>
  The ID for this model is: `flintai`
</Info>
### Modifiers

| Name          | ID           | Description                                      |
| ------------- | ------------ | ------------------------------------------------ |
| **Turbo**     | `:turbo`     | Superfast Inference _(upto 2000 tps)_            |
| **Reasoning** | `:reasoning` | With CoT Reasoning _(slower, but more powerful)_ |

### Use Cases

1. More difficult mathematical problem solving
2. Simple coding tasks
3. Decision support
4. Creative writing and storytelling
5. Role-playing and character development
