## Get All Available Models

Available models can be listed through the FlintAI Models API endpoint. You can use the `https://ai.flintstudios.workers.dev/v1/models` endpoint to return a JSON list of all active models:

<CodeGroup>

```bash Terminal
curl -X GET "https://ai.flintstudios.workers.dev/v1/models" \
     -H "Authorization: Bearer $FLINTAI_API_KEY" \
     -H "Content-Type: application/json"
```


```python Python
from flintai import FlintAI

client = FlintAI()

for model in client.models():
    print(" - " + model.name)
```

</CodeGroup>
