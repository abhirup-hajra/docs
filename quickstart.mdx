## **Requesting your first Chat Completion**

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST "https://ai.flintstudios.workers.dev/v1/chat/completions" \
         -H "Authorization: Bearer $FLINTAI_API_KEY" \
         -H "Content-Type: application/json" \
         -d '{"messages": [{"role": "user", "content": "Explain the importance of fast language models."}], "model": "flintai-mini"}'
    ```
  </Tab>
  <Tab title="Python">
    Execute this command to install the Python FlintAI SDK

    ```shellscript
    pip install flintai
    ```

    Query `flintai-mini` using ChatCompletionsAPI

    ```python
    from flintai import FlintAI
    
    ai = FlintAI(api_key="fskey-HnGs6f4b...your-api-key")
    response = ai.complete(messages="Explain the importance of fast language models.", model="flintai-mini")
    
    print(response.content)
    ```
  </Tab>
</Tabs>

## Using third-party SDKs

<Tabs>
  <Tab title="OpenAI Python SDK">
    Execute this command to install the OpenAI Python SDK.

    ```shellscript
    pip install openai
    ```
  </Tab>
</Tabs>

Query `flintai-mini` using ChatCompletionsAPI.

```python
from openai import OpenAI
import os

client = OpenAI(
    base_url="https://ai.flintstudios.workers.dev/v1",
    api_key=os.environ.get("FLINTAI_API_KEY"), # Or hardcode your api key here
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "Explain the importance of fast language models.",
        }
    ],
    model="flintai-mini",
)

print(chat_completion.choices[0].message.content)
```
